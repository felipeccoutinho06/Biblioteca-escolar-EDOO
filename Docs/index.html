📚 Sistema de Biblioteca Escolar

🔍 Sobre o Projeto

Este projeto foi desenvolvido para a disciplina de Programação Orientada a Objetos (EDOO), com o objetivo de simular um sistema de biblioteca escolar usando C++. Ele permite o cadastro de usuários e livros, controle de empréstimos e devoluções, com regras específicas para alunos e professores.

🧠 Funcionalidades

  • Cadastro de livros e usuários (alunos e professores)
  • Controle de empréstimos e devoluções
  • Regras diferentes de limite por tipo de usuário (3 para alunos, 5 para professores)
  • Validação contra empréstimos duplicados e falta de exemplares
  • Exibição de relatórios de status da biblioteca

⚙️ Como executar

Para rodar o sistema, basta digitar no terminal:

.\biblioteca.exe
