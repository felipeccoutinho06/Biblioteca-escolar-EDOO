Sistema de Biblioteca Escolar

📌 Descrição

Este projeto foi desenvolvido como parte da disciplina de Programação Orientada a Objetos em C++. O sistema permite gerenciar livros, usuários (alunos e professores), realizar empréstimos, devoluções e calcular multas.

🛠️ Funcionalidades principais

  • Cadastro de livros e usuários
  • Empréstimos e devoluções de livros
  • Controle de disponibilidade
  • Cálculo de multas por atraso
